## Project Overview
CampusConnect is a secure, anonymous chat platform designed exclusively for college students. The platform allows students with valid college email IDs to log in and engage in random one-on-one chat sessions with fellow students. The main goal is to foster social interaction, networking, and fun connections within the college community while maintaining privacy and anonymity.

## Tech Stack
* **Frontend:** React.js, Vite, Tailwind CSS
* **Backend:** Spring Boot, WebSocket, Spring Security
* **Database:** PostgreSQL
* **Authentication:** College Email Verification
* **Deployment:** Frontend on Vercel/Netlify, Backend on Render

## Features
- [ ] Anonymous Login with College Email
- [ ] Real-time Random Matchmaking
- [ ] One-on-One WebSocket Chat
- [ ] User Reporting & Blocking
- [ ] Time-limited Sessions
- [ ] Clean, Responsive UI
